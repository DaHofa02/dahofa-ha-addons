# Checkmate Home Assistant Add-on Repository

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FDaHofa02%2Fdahofa-ha-addons)

## Add-ons in this repository

### Checkmate – Uptime Monitor

[Checkmate](https://github.com/bluewave-labs/checkmate) is an open-source uptime and infrastructure monitoring tool.

Features:
- Monitor websites, APIs, and services
- Beautiful dashboard with real-time status
- Incident management & alerts
- Built-in MongoDB (no external DB needed)

## Installation

1. Navigate to **Settings → Add-ons → Add-on Store** in Home Assistant.
2. Click the three dots (⋮) → **Repositories**.
3. Add this repository URL: `https://github.com/DaHofa02/dahofa-ha-addons`
4. Find **Checkmate** in the store and click **Install**.
